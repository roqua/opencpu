# Roqua::OpenCPU

Roqua wrapper for the OpenCPU REST API.

## Installation

Add this line to your application's Gemfile:

    gem 'roqua-opencpu'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install roqua-opencpu
